Cohort Calendar

View all cohorts

This view shows all cohort data, including: cohort name, start date, graduation date, program type, campus, graduation, type, and contact hours.

View all holidays

This view shows all holidays, listed by holiday name and date. These dates have no class hours, and inform the cohort graduation dates.

View cohort calendar

This view shows a gantt chart of all cohorts filtered by campus, program, or all. This allows a view of cohort overlaps.
